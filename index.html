Rails on Web 介绍.

这是一个Ruby on Rails的定制产品，用于满足一般中小型网站的基本架构，项目集成了一些常用的功能，免去了开发中小型非功能型网站每次都从零开始scaffold的过程。

  1. download git srouce
  2. bundle install
  3. modify config/site_map.yml
  4. rake 
    db:migrate
    rails_on_web:init_all_data
    rails_on_web:generate_all_page
  5. rails server

功能特性

  • 新闻中心（新闻分类、新闻发布）
  • 留言系统（基本客户留言、留言后邮件提醒）
  • 产品管理（产品类别、产品添加）
  • 页面管理（关于我们、版权声明、联系我们等网站特别页面的创建与管理）
  • 资源管理（上传图片、视频等)
  • 后台程序（集成ActiveAdmin)
